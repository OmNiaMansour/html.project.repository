Java Script

JavaScript was originally developed by Brendan Eich. Then, it was initially introduced in the Netscape Corporation in 1995. JavaScript is a scripting or programming language for use in modification in HTML pages. It breathes life into website making it more lively, interactive, ands user-friendly. It is the third layer of the layer cake of standard web technologies. It is high-level, prototype-based object-oriented programming language. Their programs are run by an interpreter built into the user's web browser.

Some of its Applications are:

  • Games
  • Mobile Applications
  • Smartwatch Applicationss
  • Flying Robots
  • Web Servers
  • Art
  • Web Development
  • Server Applications


Major organizations that use Java Script are: WordPress, Linkedin, Soundcloud, Khan Academy, Yahoo...etc.
